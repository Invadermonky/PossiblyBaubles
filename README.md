## Possibly Baubles
Possibly Baubles is an addon to Actually Additions that adds Bauble capabilities to numerous items. The necessary code is added via mixins, allowing seamless integration as well without the need for new items.

## Features
- **Batteries** - All battery variants can be equipped as a charm
- **Engineer's Goggles** - Engineer's Goggles can be equipped as a head bauble
- **Potion Rings** - Potion Rings ccan be equipped as rings \*
- **Ring of Growth** - Ring of Growth can be equipped as a ring
- **Ring of Liquid Banning** - Ring of Liquid Banning can be equipped as a ring
- **Ring of Magnetism** - Ring of Magnetism can be equipped as a ring
- **Traveler's and Void Sacks** - Sacks can be equipped as a belt bauble \*\*
- **Wings of the Bat** - Wings of the Bat can be equipped as body bauble

\* By default only advanced rings provide their effects when equipped. This setting can be changed using the configuration.

\*\* Equipped sacks can be accessed using an all new keybind that will open their associated inventory.

## FAQ
**Q:** Doesn't Actually Baubles already exist? Why did you make this then?
**A:** Actually Baubles creates entirely new items that have to be converted from Actually Addition items. Possibly Baubles uses mixins to add the necessary code to items directly, so there is no need for the new items.
